daiaItem
directive in module ng-daia

Description

This directive displays a DAIA item, given as JSON object with parameter daia-item. The item is injected into the template's scope as item. The template template/daia-item.html can be changed with the template-url parameter.

See also daiaAvailability directive.

Usage

as attribute
<ANY daia-item
     daia-item="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
daia-itemstring

The DAIA item to display

template-urlstring

Custom template URL to display daia result

daiaAvailability
directive in module ng-daia

Description

This directive displays the availability of a DAIA item, given as JSON object with parameter daia-availability. This directive is used by the default template of directive daiaItem. The default template template/daia-availability.html of this directive can be changed with the template-url parameter.

Scope

The DAIA item is injected into the template's scope as variable item. For easier access, its members available and unavailable are provided as well, unless they are empty.

Source code

The most recent source code of this directive is available at GitHub.

Usage

as attribute
<ANY daia-availability
     daia-item="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
daia-itemstring

The DAIA item to display

template-urlstring

Custom template URL to display daia result

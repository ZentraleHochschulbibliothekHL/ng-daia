daiaSimple
filter in module ng-daia

Description

This filter can be used to transform a DAIA response, document, or item into simple availability status (DAIA Simple). The filter returns a plain object with simple key-value pairs, such as:

{ status: "openaccess" }
{ status: "loan" }
{ status: "presentation" }
{ status: "expected" }
{ status: "expected", expected: "2014-12-07" }
{ status: "openaccess", href: "http://dx.doi.org/10.1901%2Fjaba.1974.7-497a" }
{ status: "none" }

The filter is used by directive daiaSimple. It can also be used with directive daiaApi to query and display an availability status.

Source code

The most recent source code of this filter is available at GitHub.

Usage

In HTML Template Binding

{{ daiaSimple_expression | daiaSimple }}

In JavaScript

$filter('daiaSimple')()

daiaSimple
filter in module ng-daia

Description

This filter can be used to transform a DAIA response, document, or item into simple availability status (DAIA Simple). The filter returns a simple object with simple key-value pairs, such as:

{ status: "openaccess" }
{ status: "loan" }
{ status: "presentation" }
{ status: "expected" }
{ status: "expected", expected: "2014-12-07" }
{ status: "openaccess", href: "http://dx.doi.org/10.1901%2Fjaba.1974.7-497a" }
{ status: "none" }

The filter can also be used with parameter daia-filter at directive daiaApi.

To customize the message, use angular-translate and the translate directive.

Usage

In HTML Template Binding

{{ daiaSimple_expression | daiaSimple }}

In JavaScript

$filter('daiaSimple')()

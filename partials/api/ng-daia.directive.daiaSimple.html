daiaSimple
directive in module ng-daia

Description

This directive displays a DAIA response, document, or item in simplified form, as filtered by daiaSimple). The default template template/daia-simple.html can be changed with the template-url parameter. The template's scope gets the variables

  • daia: DAIA response, document, or item as passed to the directive
  • status: simplified status (set to none by default)
  • expected, delay, href, limitation: optional additional information

Usage

as attribute
<ANY daia-simple
     daia-item="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
daia-itemstring

DAIA response, document, or item to display

template-urlstring

Custom template URL to display daia result

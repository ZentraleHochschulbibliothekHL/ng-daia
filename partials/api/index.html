ng-daia - Documentation

ng-daia is an AngularJS module to facilitate access to Document Availability Information API (DAIA) services and to display the responses given by a DAIA server. See ng-daia for an introduction of usage.

Source code and development

The module is managed in a public git repository at https://github.com/gbv/ng-daia. Please use https://github.com/gbv/ng-daia/issues for bug reports and feature requests.

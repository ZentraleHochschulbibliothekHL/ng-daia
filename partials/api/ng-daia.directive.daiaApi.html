daiaApi
directive in module ng-daia

Description

This directive queries a DAIA server, each time one of its parameters daia-api or daia-id is changed. The DAIA response (optionally filtered by a filter such as daiaSimple) is injected into the template's scope as variable daia.

The default template template/daia-response.html makes use of the directives daiaItem and daiaAvailability. The template can be changed with the template-url parameter.

Usage

as attribute
<ANY daia-api
     daia-api="{string}"
     daia-id="{string}"
     daia-filter="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
daia-apistring

Base URL of DAIA server to query from

daia-idstring

Document identifier to query for

daia-filterstring

AngularJS filter to process daia response, e.g. daiaSimple

template-urlstring

Custom template to display DAIA result

daiaApi
directive in module ng-daia

Description

This directive queries a DAIA server, each time one of its parameters daia-api or daia-id is changed. The DAIA response is injected into the template scope as daia. The template template/daia-response.html can be changed with the template-url parameter.

The default template makes use of the directives daiaItem and daiaAvailability.

Usage

as attribute
<ANY daia-api
     daia-api="{string}"
     daia-id="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
daia-apistring

Base URL of DAIA server to query from

daia-idstring

Document identifier to query for

template-urlstring

Custom template to display DAIA result
